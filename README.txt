# AnimationProject

Created By Lalit Mada, Arvind Dhaliwal, Bhavjot Gill, Edison Liang

This program creates an animation with a varying shapes moving across a screen. This is done by creating a window using javaswing,
and then taking a shape choice by txt file of either "circle","rectangle", or "line".

The program will create the animation window by simply running the program and letting the function run.
